# Gazetkowo — podgląd

Prototyp mobilnej listy zakupów (statyczny HTML/CSS/JS, bez build stepu).

## Uruchomienie lokalnie
Wystarczy otworzyć `index.html` w przeglądarce, albo:

```bash
npx serve .
```

## Deploy na Vercel
- Framework Preset: **Other**
- Build Command: *(puste)*
- Output Directory: *(puste / root)*
